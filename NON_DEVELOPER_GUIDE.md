# Detective Voice — 비개발자 창업자 가이드

> 이 문서는 코드 한 줄 안 짜본 사람이 이 프로젝트를 굴려 iOS 앱스토어 출시까지 가는 길을 정리한 거예요.
> 두 가지를 답해요. (1) 지금 받은 결과물이 각각 무엇인지, (2) iOS 비즈니스화까지 12~18개월 어떻게 가는지.

---

## Part 1 — 지금 받은 결과물, 어떻게 봐야 하나요?

### 🎬 가장 먼저: `demo.html`을 더블클릭하세요

`detective-voice-prototype/demo.html` 파일을 **그냥 더블클릭**하면 브라우저(Chrome, Safari)에서 열려요. 코드 실행 같은 거 필요 없어요. 클릭만으로 7분짜리 사건 한 편을 처음부터 끝까지 체험할 수 있어요. 🔊 버튼을 누르면 Aria의 음성도 들려줘요.

이게 **게임이 어떻게 작동하는지 눈으로 보는 가장 빠른 방법**이에요. 다른 파일들은 이 데모를 진짜 게임으로 만드는 데 쓰는 재료들이에요.

### 📂 다른 파일들의 정체

각 파일을 영화 제작에 비유하면 이렇게 돼요:

| 파일 | 영화로 비유하면 | 쉽게 말해 |
|---|---|---|
| `demo.html` | 시네마틱 트레일러 | 클릭으로 체험하는 데모. 비개발자에게 보여줄 때 1순위 |
| `cases/last_customer.json` | **시나리오 (대본)** | 사건의 모든 진실 — 범인·동기·NPC 5명·단서 10개·결말 4가지가 전부 적혀 있어요. 이 게임은 작가가 이런 시나리오를 매주 한 편씩 쓰는 게 핵심이에요. |
| `prompts/aria_gm.md` | **연출자에게 주는 디렉션 노트** | AI(Aria)에게 "이런 톤으로, 이런 식으로 진행해라"고 가르치는 설명서 |
| `prompts/character.md` | **배우에게 주는 캐릭터 분석** | NPC 한 명 한 명을 AI가 어떻게 연기할지 가르치는 템플릿 |
| `schemas/case_template.schema.json` | **시나리오 작가용 양식** | 새 사건을 쓸 때 빠뜨리지 않아야 할 항목 목록 (체크리스트 같은 것) |
| `detective_voice.py` | **상영관 (영사기)** | 시나리오와 연출 노트를 받아 AI한테 넣고 실제로 사건을 굴리는 엔진. 개발자가 손볼 부분. |
| `PROTOTYPE_ROADMAP.md` | **제작 일정표** | 4~6주 어떤 순서로 검증할지 |
| `README.md` | **개발자용 사용 설명서** | `detective_voice.py`를 굴리는 방법. 비개발자는 안 봐도 됨. |

### 💡 핵심 통찰

당신이 직접 손대야 할 파일은 **`cases/last_customer.json` 같은 사건 시나리오**예요. 코드는 한 번 만들어두면 외주 개발자가 유지보수하지만, **사건은 당신이 매주 또는 매월 새로 써내야 해요.** 그게 이 게임의 컨텐츠이자 차별화이자 영혼이에요.

JSON 파일은 무서워 보이지만, 사실 그냥 **구조화된 표 같은 거**예요. 메모장이나 [VS Code](https://code.visualstudio.com/) 같은 무료 에디터로 열어서 글자만 바꾸면 돼요. 처음엔 외주 작가/PD가 작성하고, 익숙해지면 직접 쓸 수 있어요.

## Part 2 — iOS 출시까지의 길

### 큰 그림: 4 페이즈, 12~18개월

```
[Phase A: 검증]      [Phase B: MVP]      [Phase C: 베타]     [Phase D: 출시]
   2~3개월             4~6개월              3~4개월             2~3개월
   - 데모 검증          - iOS 앱 빌드        - 클로즈드 알파      - 앱스토어 심사
   - 시각장애 사용자     - 사건 3편 완성       - 한국 200명 베타     - 한국 출시
     5~10명 인터뷰       - AI 백엔드 안정화    - 가격·전환 검증      - 글로벌 영어판
   - 공동창업자/팀       - 시각장애 사용자      - 콘텐츠 시즌 운영     - PR·인플루언서
     초기 영입            UX 통과
```

### Phase A: 검증 (지금 ~ 2~3개월)

**목적**: 게임이 정말 재밌고 풀 수 있는지, 시각장애 사용자가 실제로 좋아하는지를 돈 적게 들여서 확인.

**할 일 (매주 단위)**

1. **이번 주**: `demo.html`을 5명에게 보여줘 — 게임 디자이너 친구 1명, 시각장애 사용자 1~2명, 일반 사용자 2명. "이거 출시되면 사겠나?" 물어봐요. 반응을 노트에 적어요.

2. **2~4주차**: `detective_voice.py`(텍스트 프로토타입)를 외주 개발자 1명에게 의뢰해서 본인 컴퓨터나 웹에서 친구가 돌려볼 수 있게 만들어요. 비용 ₩50~150만원 (1~2주 작업). [크몽](https://kmong.com/), [이랜서](https://www.elancer.co.kr/), 또는 X(트위터)에서 "AI 게임 프로토타입 백엔드 1주 외주" 같은 식으로 모집.

3. **5~8주차**: 시그니처 사건 1편을 완성된 형태로 다듬고, **시각장애 사용자 인터뷰 5~10명**을 진행해요. 한국시각장애인연합회, 한빛맹학교, 점자도서관 같은 곳에 협조 요청. Dot Inc 사용자 네트워크가 있다면 그게 1순위.

4. **9~12주차**: 위 인터뷰 결과를 정리해서 **공동창업자 또는 핵심 엔지니어를 영입**. 또는 외부 펀딩 신청 준비.

**예산**: ₩500만원 ~ ₩1,500만원 (외주 개발 + 인터뷰 사례비 + AI API 비용)

**Phase A 통과 기준**:
- 시각장애 사용자 5명 중 4명이 "출시되면 다운로드하겠다"고 답함
- 사건 1편이 30~50분에 풀리고 NPS 50 이상
- 공동창업자 또는 CTO 후보 1명 확보, 또는 시드 펀딩 LOI 1건

### Phase B: MVP (4~6개월)

**목적**: 진짜 iOS 앱으로 빌드. 시그니처 사건 3편 + procedural 사건 무한 + 베타 배포 준비.

**할 일**

1. **팀 구성** (최소 인원):
   - **iOS 개발자 1명** (정직원 또는 6개월 계약직). 음성 입력, TTS 통합, 앱스토어 빌드까지.
   - **AI/백엔드 개발자 1명** (프로토타입 만든 사람이 그대로 가는 게 베스트).
   - **사운드 디자이너 1명** (프리랜서 가능). BGM, 환경음, 효과음.
   - **사건 작가 1~2명** (외주 또는 본인). 시나리오 3편 + procedural 풀.
   - **본인 = 디렉터/PM**.

2. **iOS 개발자 영입 옵션**:
   - **사내 영입**: Dot Inc에 iOS 엔지니어가 있다면 1순위. Dot Pad 연동까지 자연스러움.
   - **외주 에이전시**: 한국 기준 월 ₩800만~1,500만원/명, 6개월이면 ₩5천만~1억. 빠르지만 비싸고 IP 관리 주의.
   - **공동창업자/CTO**: 지분 5~15%로 영입. 가장 싸고 가장 강하지만 사람 찾기가 가장 어려움. [Founders Network 코리아](https://www.foundersnetwork.kr/), 카카오/네이버 출신 게임 엔지니어 커뮤니티, AngelList Korea 등.

3. **iOS 앱 빌드의 실제 단계**:
   - [Apple Developer Program](https://developer.apple.com/programs/) 가입 — **연 $99 (~₩13만원)**. 회사 명의 가입 권장 (D-U-N-S 번호 필요, 발급 무료).
   - Mac 1대 필수 (Xcode 빌드용). 없으면 MacBook Air M2 ₩140만원이 가장 저렴.
   - 앱 아이콘, 스크린샷, 앱 스토어 설명, 개인정보처리방침 페이지 준비.

4. **AI 비용 가드**: GDD가 1사건당 $0.30~0.60 예상. 베타 100명이 월 5사건 플레이하면 월 $200 정도. 출시 후 10,000명이면 월 $20,000~30,000 — 이 시점에 자체 모델 fine-tune 또는 캐싱으로 50% 절감 필요.

**예산**: ₩1.5억 ~ ₩4억
- 인건비 5~6개월: ₩1억 ~ ₩2.5억
- AI/클라우드: ₩500만 ~ ₩1,500만
- 사운드/성우: ₩2천만 ~ ₩5천만
- 기타: ₩1천만 ~ ₩3천만

### Phase C: 베타 (3~4개월)

**할 일**

1. **TestFlight 클로즈드 알파** (시각장애 사용자 30명, 4주). 피드백 정리 → 핵심 이슈 수정.
2. **TestFlight 오픈 베타** (한국 200~500명, 8주). 가격 모델·전환율 검증. ₩9,900짜리 프리미엄 vs ₩4,900 월 구독 A/B 테스트.
3. **콘텐츠 시즌 운영 리허설**: 매주 procedural 사건 1편씩 자동 생성 → QA → 배포 파이프라인 점검.
4. **언론·인플루언서 컨택**: 게임 미디어(인벤, 디스이즈게임), 시각장애 커뮤니티 인플루언서, 팟캐스트 진행자.

### Phase D: 출시 (2~3개월)

1. **앱스토어 심사 제출** — 보통 1~3일, 거부 시 사유 받고 재제출. 음성 게임은 마이크 권한 사용 사유 명확히 기재 필수.
2. **한국 출시 → 영어판 출시** (한 달 간격 권장).
3. **PR**: 보도자료, "장애인 접근성을 정조준한 K-콘텐츠 게임" 앵글로 일반 매체 어필.
4. **Apple Featured 노림수**: Apple Korea의 접근성팀에 사전 컨택. WWDC 시즌(6월)이나 Global Accessibility Awareness Day(5월 셋째 주 목요일) 타이밍 노리면 Featured 가능성 ↑.

### 출시 후 12개월 KPI (GDD 기준)

- 다운로드 150,000+ (글로벌)
- 유료 전환율 8%+
- 연 매출 USD 800K~1.5M (₩10억~20억)

## Part 3 — 펀딩 옵션 (한국 기준)

### 자체 자금 (가장 간단)

Phase A는 ₩500만~1,500만원이면 충분해요. 본인 자금이나 Dot Inc 사내 R&D 예산으로 가능. **이걸 먼저 쓰는 게 맞아요** — 외부 펀딩은 검증된 결과를 갖고 들어갈 때 조건이 훨씬 좋아져요.

### 정부 지원 (비희석, 가장 추천)

| 프로그램 | 규모 | 핵심 |
|---|---|---|
| **KOCCA 게임 제작 지원** | ₩1억~5억 | 한국콘텐츠진흥원. 매년 2~3회 공고. 시각장애 게임 + K-콘텐츠 결합은 강점. |
| **TIPS** | ₩5억 + 후속 ₩4억 | 운영사를 통해 신청. 게임은 통과 어려운 편이지만 'AI + 접근성'은 가능성 있음. |
| **장애인기업종합지원센터** | ₩2천만~1억 | 장애인 사용자 대상 서비스라 적합. |
| **K-Global 300** | ₩3천만~1억 | 글로벌 진출 장려. |
| **서울시 게임콘텐츠센터** | 인큐베이션 + ₩수천만 | 강남 사무실 + 멘토링 + 자금. |

### VC/액셀러레이터

| 단계 | 규모 | 추천 대상 |
|---|---|---|
| 시드 | ₩3억~10억 | Krafton 액셀러레이터, 본엔젤스, 카카오벤처스, 매쉬업엔젤스 |
| 시리즈 A | ₩20억~50억 | 알토스벤처스, 캡스톤파트너스 (게임 트랙 있음) |

**팁**: 게임 VC는 "출시 후 매출"을 보지만, "AI 게임 + 접근성"은 일반 SaaS VC가 더 환영해요. 카테고리를 잘 잡으면 밸류에이션이 달라져요.

### 크라우드펀딩 (검증 + 자금 동시)

- **와디즈** — 한국, 게임 앞결제로 시드 펀딩 + 마케팅 동시. 시각장애 접근성 스토리는 와디즈 정서에 잘 맞아요.
- **Kickstarter** — 글로벌, 영어판 출시 직전에 권장. 음성 게임은 비주얼이 약해서 영상 퀄리티가 핵심.

## Part 4 — 팀 빌딩 의사결정

### 지금부터 첫 6개월, 누구를 어떻게 모을까?

| 역할 | 우선순위 | 형태 | 비용 |
|---|---|---|---|
| **iOS 엔지니어** | 1순위 | 풀타임 정직원 또는 공동창업자 | ₩6천만~1.2억/년 또는 지분 5~10% |
| **AI/백엔드 엔지니어** | 1순위 | 풀타임 또는 6개월 계약직 | ₩6천만~1억/년 |
| **사운드 디자이너** | 2순위 | 프리랜서, 사건당 작업 | 사건 1편 ₩500만~2,000만원 |
| **사건 작가** | 2순위 | 외주 (드라마 작가, 추리소설 작가) | 사건 1편 ₩300만~1,000만원 |
| **시각장애 UX 컨설턴트** | 1순위 | 시간제 (월 8~16시간) | 시간당 ₩5만~15만원 |
| **성우 (Aria)** | 3순위 | 외주, 작품 단위 | 한국어 ₩300만~1,500만원, 영어 ₩500만~3,000만원 |

**가장 중요한 결정**: 1순위 두 자리(iOS + AI 백엔드)를 **공동창업자로 갈 것인가, 정직원으로 갈 것인가**. 공동창업자는 지분이 들지만 헌신과 속도가 다르고, 정직원은 깔끔하지만 1년 내 이탈 리스크가 있어요. Dot Inc 같은 모회사가 있다면 사내 발탁이 가장 안정적.

### 공동창업자/CTO 영입 — 어디서 만나나

- **AI Builders KR**, **GDG Seoul** 같은 기술 커뮤니티 — Slack/디스코드 들어가서 친해지기.
- **카이스트/포항공대 게임 동아리**, **네오위즈/넥슨/펄어비스 출신 엔지니어**.
- **트위터(X) 한국 AI 커뮤니티** — 의외로 공동창업자가 많이 만나져요.
- **Dot Inc 내부** — 가장 빠르고 안전한 길. Dot Pad 연동까지 자연스러움.

영입의 핵심은 "이 게임의 비전"이 매력적이라는 거예요. **"화면 없이 즐기는 추리극, AI가 무한 사건을 만드는 게임, 시각장애 사용자가 처음으로 갖게 될 주류 게임"** — 이건 엔지니어한테 거절하기 어려운 비전이에요.
